## ✨ Особенности

**Time Schedule Platform** — это комплексная система для управления временем и личностного роста, разработанная с фокусом на мотивацию и саморефлексию.

### 🏠 **Главная Dashboard**
- Обзор дня с задачами, целями и достижениями
- Быстрая статистика продуктивности
- Мотивационные элементы и цитаты дня
- Быстрые действия для всех разделов

### 📅 **Интерактивный календарь**
- Месячный, недельный и дневной виды
- Создание событий и планирование задач
- Цветовая кодировка по категориям
- Интеграция с системой задач и целей

### ✅ **Система ежедневных задач**
- Управление задачами с приоритетами
- Категории: работа, личное, здоровье, обучение, хобби
- Drag & drop для изменения порядка
- Трекинг привычек и статистика выполнения

### 🎯 **Планирование целей**
- SMART цели на разные периоды (год, квартал, месяц, неделя)
- Разбивка целей на вехи с трекингом прогресса
- Визуализация достижений
- Категоризация по сферам жизни

### 🏆 **Коллекция достижений**
- Личная галерея успехов для мотивации
- Категории: карьера, образование, здоровье, творчество
- Эмоциональная рефлексия и анализ влияния
- Случайное достижение дня для вдохновения

### 🤔 **Работа с собой**
- **Неуверенности**: честная работа со страхами и сомнениями
- **Рефлексия**: ежедневный анализ эмоций и достижений
- **Личностный рост**: планирование развития навыков

## 🚀 Быстрый старт

### Требования
- Node.js 18.x или выше
- npm или yarn

### Установка

1. **Клонирование репозитория:**
```bash
git clone [repository-url]
cd time-schedule-platform
```

2. **Установка зависимостей:**
```bash
npm install
```

3. **Запуск в режиме разработки:**
```bash
npm run dev
```

4. **Открыть в браузере:** [http://localhost:3000](http://localhost:3000)

## 🎨 Дизайн и UX

### **Принципы дизайна**
- **Minimalist & Clean** — чистый интерфейс без отвлекающих элементов
- **Мотивирующий** — позитивные цвета, празднование успехов
- **Персональный** — кастомизация и личные данные
- **Responsive** — работает на всех устройствах

### **Цветовая схема**
- 🟢 **Успех** — зеленые тона для выполненных задач
- 🔵 **Прогресс** — синие тона для текущих целей
- 🟡 **Внимание** — желтые тона для важных задач
- 🟣 **Рефлексия** — фиолетовые тона для самоанализа

### **Геймификация**
- Streak counters для привычек
- Progress rings для целей
- Achievement unlock анимации
- Ежедневный рейтинг продуктивности

## 🛠 Технологический стек

### **Frontend**
- **Next.js 14** — React фреймворк с App Router
- **TypeScript** — типизация для надежности
- **Tailwind CSS** — utility-first CSS фреймворк
- **shadcn/ui** — высококачественные UI компоненты
- **Framer Motion** — библиотека анимаций
- **Lucide React** — красивые SVG иконки

### **Особенности**
- Темная/светлая тема с переключением
- Анимации на скролл с blur эффектами
- Responsive дизайн для всех устройств
- TypeScript для type safety
