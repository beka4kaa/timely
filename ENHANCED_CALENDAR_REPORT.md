## 📖 Инструкция по использованию

### Первый запуск:

1. **Вход в систему**:
   - Перейдите на `http://localhost:3000`
   - Система автоматически перенаправит на страницу входа
   - Используйте тестовый аккаунт или создайте новый

2. **Навигация**:
   - После входа откроется дашборд с боковой панелью
   - Календарь доступен в разделе "Календарь и планирование"

### Работа с календарем:

1. **Просмотр событий**:
   - Переключайтесь между режимами: месяц/неделя/день
   - Используйте навигацию для перехода между периодами
   - Клик на событие открывает детальную информацию

2. **Создание события**:
   - Нажмите кнопку "Добавить" в правом верхнем углу
   - Заполните обязательные поля (название)
   - Настройте время, категорию, приоритет
   - Добавьте дополнительную информацию по желанию

3. **Поиск и фильтрация**:
   - Используйте поле поиска для нахождения событий
   - Применяйте фильтры по типу события
   - Результаты обновляются в реальном времени

4. **Управление событиями**:
   - Клик на событие для просмотра деталей
   - Кнопка редактирования (в разработке)
   - Кнопка удаления для удаления события

## 🔧 Техническая архитектура

### Frontend:
- **Next.js 14** с App Router
- **TypeScript** для типобезопасности
- **Tailwind CSS** для стилизации
- **shadcn/ui** для компонентов
- **Lucide React** для иконок
- **date-fns** для работы с датами

### Backend:
- **Next.js API Routes** для серверной логики
- **RESTful API** с полной CRUD функциональностью
- **Cookie-based auth** для сессий
- **Middleware** для защиты маршрутов

### Состояние:
- **React Context** для управления аутентификацией
- **useState/useEffect** для локального состояния
- **API интеграция** с обработкой ошибок
