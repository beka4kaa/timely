## 📊 СХЕМА БАЗЫ ДАННЫХ (PRISMA)

### User-Centric Data Models:
- **Goal**: Цели пользователя с приоритетами и статусами
- **Task**: Задачи связанные с целями 
- **Achievement**: Достижения пользователя
- **UserSettings**: Настройки и предпочтения пользователя
- **Event**: События календаря пользователя

Все модели имеют `userId` с каскадным удалением при удалении пользователя.

## 🔗 API ENDPOINTS

### ✅ Goals API (`/api/goals/`)
- **GET** `/api/goals` - Получить все цели пользователя
- **POST** `/api/goals` - Создать новую цель
- **GET** `/api/goals/[id]` - Получить конкретную цель
- **PUT** `/api/goals/[id]` - Обновить цель
- **DELETE** `/api/goals/[id]` - Удалить цель

### ✅ Tasks API (`/api/tasks/`)
- **GET** `/api/tasks` - Получить все задачи пользователя
- **POST** `/api/tasks` - Создать новую задачу
- **GET** `/api/tasks/[id]` - Получить конкретную задачу
- **PUT** `/api/tasks/[id]` - Обновить задачу
- **DELETE** `/api/tasks/[id]` - Удалить задачу

### ✅ Achievements API (`/api/achievements/`)
- **GET** `/api/achievements` - Получить достижения пользователя
- **POST** `/api/achievements` - Создать достижение

### ✅ User Settings API (`/api/user/settings/`)
- **GET** `/api/user/settings` - Получить настройки пользователя
- **PUT** `/api/user/settings` - Обновить настройки

### ✅ User Profile API (`/api/user/profile/`)
- **GET** `/api/user/profile` - Профиль пользователя
- **PUT** `/api/user/profile` - Обновить профиль

### ✅ User Statistics API (`/api/user/stats/`)
- **GET** `/api/user/stats` - Статистика пользователя

### ✅ Events API (`/api/events/`)
- **GET** `/api/events` - События календаря пользователя
- **POST** `/api/events` - Создать событие
- **GET** `/api/events/[id]` - Получить событие
- **DELETE** `/api/events/[id]` - Удалить событие
