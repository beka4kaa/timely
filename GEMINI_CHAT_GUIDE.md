# Gemini AI Chat Integration

## 🚀 Новая функция: AI Помощник по планированию!

В приложение добавлен интеллектуальный чат с Gemini AI, который поможет вам улучшить планирование и продуктивность.

### 🎯 Возможности:

- **Анализ расписания**: AI видит ваше текущее расписание и может предложить улучшения
- **Советы по продуктивности**: Получите персональные рекомендации
- **Планирование времени**: Помощь в организации задач и событий
- **Контекстные ответы**: AI понимает что вы видите на экране

### 🔧 Настройка:

1. **Получите API ключ Gemini**:
   - Перейдите на [ai.google.dev](https://ai.google.dev)
   - Создайте новый проект
   - Получите API ключ для Gemini Pro

2. **Активируйте чат**:
   - Нажмите на фиолетовую кнопку с искрами в правом нижнем углу
   - Введите ваш API ключ
   - Начните общение!

### 💡 Примеры вопросов:

- "Как улучшить моё расписание?"
- "Проанализируй мою продуктивность"
- "Какие перерывы лучше добавить?"
- "Как лучше распределить задачи по времени?"
- "Посоветуй оптимальное время для встреч"

### 🎨 Особенности интерфейса:

- **Минималистичный дизайн**: Не отвлекает от основной работы
- **Плавающее окно**: Можно свернуть/развернуть
- **Адаптивность**: Красиво выглядит на любом экране
- **Быстрые кнопки**: Готовые вопросы для начала
- **Автосохранение**: API ключ сохраняется локально

### 🔒 Безопасность:

- API ключ хранится только в вашем браузере
- Никакие данные не отправляются на сторонние серверы
- Прямое подключение к Google Gemini API

### 🎪 Как использовать:

1. Откройте любую страницу приложения
2. Нажмите на кнопку чата (фиолетовая с искрами)
3. Введите API ключ при первом запуске
4. Задавайте вопросы и получайте умные советы!

AI видит ваше текущее расписание и может дать персональные рекомендации для повышения продуктивности! 🚀
